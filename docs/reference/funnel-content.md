# Funnel content (PDF 1:1)

**Джерело:** `Карта с фигурными скобками.pdf` (a.k.a. `skkk.pdf`).

Цей файл є **джерелом правди** для початкового seed-у воронки. Будь-які зміни тексту проходять **через адмін-панель в боті**, а не через цей файл (бо БД є фактичним станом). Файл лишається як reference на оригінал PDF — щоб можна було повернути формулювання якщо ніщо не плутайт.

Емодзі-плейсхолдери з PDF (`�`) замінено на читабельні аналоги (👶, 💼, 🎯 тощо) — це **єдина свобода** від оригіналу.
